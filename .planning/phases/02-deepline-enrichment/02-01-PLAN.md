This replaces the need to integrate 5-6 individual provider APIs.

Output: Three new async functions in research_fetcher.py (`_deepline_execute`, `fetch_deepline_email`, `fetch_deepline_company`), config settings, webhook wiring, and comprehensive unit tests via TDD.
